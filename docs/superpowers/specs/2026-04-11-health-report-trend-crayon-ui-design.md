# 健康报告趋势功能 + 蜡笔美学UI风格设计

> 设计日期: 2026-04-11
> 项目: health_records Flutter健康管理APP

## 一、功能概述

### 1.1 核心功能
- **多报告管理**: 每个家人可上传多份体检报告
- **报告切换**: 在家人报告详情页可选择不同报告查看对应指标
- **趋势图展示**: 根据多份报告中相同指标项生成趋势图

### 1.2 UI风格改造
- 全面采用Storybook蜡笔美学风格
- 配色为温暖大地色系（森林绿、芥末黄、砖红、米白色）
- 运用蜡笔笔触质感、手绘歪扭边框、纸张纹理背景
- 可爱插画角色作为家庭成员头像
- 圆润童趣的字体，装饰星星贴纸

### 2.1 页面流程

## 五、趋势图详细设计

### 5.1 技术方案
- 使用 `fl_chart` 包（版本 ^0.68.0）绘制图表
- 自定义绘制器模拟蜡笔笔触效果
- 数据点标注使用星星图标或圆点

### 5.2 图表组件结构
```dart
class IndicatorTrendChart extends StatelessWidget {
  final String indicatorType;     // '血糖'/'血压'/'血脂'/'自定义'
  final List<IndicatorDataPoint> dataPoints;
}
```

### 5.3 不同指标类型的图表实现

| 指标类型 | 图表形式 | 特殊处理 |
|----------|----------|----------|
| 血糖 | 单折线图 | 一个数值，蜡笔笔触线条 |
| 血压 | 双折线图 | 收缩压+舒张压两条线，或显示为"120/80"单点 |
| 血脂 | 四折线图 | TC/TG/HDL/LDL四条线，不同颜色区分 |
| 自定义 | 单折线图 | 按指标类型名称分组显示 |

### 5.4 图表功能
1. **折线图基础**: X轴日期、Y轴数值、自动缩放
2. **数据点标注**: 每个点显示数值气泡，点击查看详情
3. **异常标记**: `isAbnormal=true` 的点用红色高亮
4. **蜡笔效果**: 线条带随机抖动，模拟手绘质感

## 六、蜡笔美学UI风格

### 6.1 配色方案（温暖大地色系）

| 颜色名称 | 色值 | 用途 |
|----------|------|------|
| 森林绿 | `#4A7C59` | 主色调、卡片背景、强调色 |
| 芥末黄 | `#E5B844` | 按钮高亮、星星、重要标记 |
| 砖红 | `#B8472A` | 警告、异常指标、删除按钮 |
| 米白色 | `#F5F1E6` | 页面背景、纸张纹理底色 |
| 深棕色 | `#3D2914` | 文字、边框、标题 |
| 淡粉色 | `#F8E4E1` | 辅助卡片、女性头像装饰 |

### 6.2 视觉元素实现

**纸张纹理背景**:
- 使用预制PNG素材：`assets/textures/paper_texture.png`
- 基础色为米白色 `#F5F1E6`
- 叠加微妙噪点纹理

**手绘歪扭边框**:
- 使用 `CustomPaint` + 随机路径算法
- 边框路径带有随机偏移（±2-5像素）
- 边框颜色：深棕色 `#3D2914`
- 边框宽度：2-3像素，带抖动效果

**蜡笔笔触质感**:
- 线条绘制使用 `StrokeCap.round`
- 线条宽度变化（2-4像素随机）
- 随机抖动点模拟蜡笔不均匀效果

**装饰星星贴纸**:
- 使用预制SVG素材：`assets/icons/star.svg`
- 金色（芥末黄 `#E5B844`）
- 用于装饰卡片、按钮、正常指标标记

### 6.3 字体方案
- 使用系统默认字体
- 通过样式调整实现童趣感：
  - `fontWeight: FontWeight.w500`（圆润感）
  - `letterSpacing: 0.5`（宽松排列）
  - 圆角按钮、卡片营造柔和感

## 十一、错误处理和边界情况

| 场景 | 处理方式 |
|------|----------|
| 家人没有报告 | 卡片显示"0份报告"，点击后显示空状态提示："暂无体检报告，点击添加" |
| 某指标只有1个数据点 | 显示单点图 + 提示："仅有一份报告，无法显示趋势，添加更多报告后可查看趋势" |
| 某指标无数据 | 显示空图表 + 提示："暂无该指标数据" |
| 报告日期相同 | X轴日期相同时，在图表上合并显示，标注"2份报告" |
| 自定义指标类型不同 | 每个自定义指标单独一条折线，类型名称作为标签 |
| 指标单位不一致 | 同类型指标默认单位相同；若不一致，显示警告并取最新报告的单位 |
